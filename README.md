# Kasperkey Toolbox

Reverse-engineering of the [Kasperkey Toolbox](https://www.52pojie.cn/thread-2018085-1-1.html) from by @huawei_518 on 52pojie.

## What works

- The tk UI.
- Reset activation.

## Requirement

> [!NOTE]
> The CPython must not be the 32-bit version,
> otherwise accessing to Wow6432Node may encounter failures.

- Windows x64
- CPython 3.8+
